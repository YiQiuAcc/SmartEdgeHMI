# SmartEdge-HMI: 工业级边缘网关监控看板

## 📖 项目简介

SmartEdge-HMI 是一个基于 **.NET 8 WPF** 构建的现代工业设备上位机与实时数字孪生监控系统。
本项目采用**软硬件解耦**的架构设计：底层通信抽象层支持快速平滑切换 **虚拟设备模拟器（Virtual COM）**与**真实物理边缘网关（Arduino/ESP8266 + 传感器）** 。系统实现了端到端的工业级数据采集、高频流数据解析、实时图表渲染与边缘本地持久化存储。

**🎯 核心业务场景：** 自动化产线/机房设备的实时遥测（Telemetry）、状态监控、双向控制下发与历史异常追溯。

## 🛠 技术栈 (Tech Stack)

- **UI 框架** ：WPF (.NET 8)
- **架构模式** ：MVVM (基于 `CommunityToolkit.Mvvm` 现代工具包)
- **底层通信** ：`System.IO.Ports` (串口 RS232 概念流处理)
- **高性能图表** ：`ScottPlot` (基于 OpenGL/Skia，支撑百万级数据点高帧率渲染，无卡顿)
- **边缘存储** ：SQLite + Dapper (轻量级本地结构化日志与配置存储)
- **工程化** ：`Serilog` (异步结构化日志，按天轮转切割)、`System.Text.Json` (零分配 JSON 反序列化)

## 🌟 核心架构亮点 (面试闪光点)

1. **异步数据流处理** ：使用独立后台 `Task` 监听串口数据流，结合 `Channel<T>` 读写队列实现生产者-消费者模式，确保 UI 线程在高频数据涌入时绝对流畅。
2. **双向协议通信** ：不仅支持被动接收硬件遥测数据（如温度、转速），还支持向硬件下发 JSON 控制指令（如修改阈值、强制复位）。
3. **防抖与异常恢复** ：内置串口断线检测、自动重连机制以及心跳包（Heartbeat）超时判定。
